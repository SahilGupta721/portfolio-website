# Sahil Gupta Portfolio

A personal developer portfolio showcasing my projects, skills, and experience as a Software Developer specializing in AI/ML and full-stack development.

## Sections

- **Projects** — Featured work including AI systems, web apps, and hackathon builds
- **About** — Background and vision as a developer
- **Skills** — Technologies and tools I work with
- **Contact** — Ways to get in touch

## Tech Stack

- [Next.js](https://nextjs.org) — React framework with file-based routing
- [React.js](https://react.dev) — UI component library
- [Tailwind CSS](https://tailwindcss.com) — Utility-first CSS framework
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — Core language

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
